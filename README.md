# Hospital Management Dashboard - Power BI

Project Overview

The Hospital Management Dashboard is an interactive Power BI project developed to analyze and visualize hospital operations, patient records, doctor information, billing performance, and disease trends. The dashboard helps healthcare organizations monitor key metrics and make data-driven decisions efficiently.

Objectives
Analyze patient admission and discharge details.
Monitor hospital billing, pending payments, and payment methods.
Track doctor specialization and department performance.
Visualize disease distribution and treatment trends.
Provide interactive filtering for better healthcare insights.
